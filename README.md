# Maison Aurèle

Site vitrine du restaurant gastronomique **Maison Aurèle** — Paris, Ier arrondissement.

Cuisine française d'auteur, entre tradition et audace contemporaine.

---

## Stack

- HTML / CSS statique
- [Vite](https://vitejs.dev/) (bundler & dev server)
- Fonts : Cormorant Garamond + Inter (Google Fonts)

## Développement local

```bash
npm install
npm run dev
```

Ouvre http://localhost:5173

## Build de production

```bash
npm run build
```

Le site optimisé est généré dans `dist/`.

## Déploiement

Déployé automatiquement via Hostinger à chaque push sur `main`.
